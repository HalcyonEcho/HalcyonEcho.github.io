Custom Python Hash Algorithm SAMPLE FORMAT

My quirky, small-scale hash algorithm with custom padding & variable input size. Fun for learning & experimentation!

Description

This project is a small, educational hash algorithm written in Python. It accepts variable-length input and applies a custom padding marker \xBE\xEE. Designed for experimentation, it produces at least 20-bit outputs.

Features

  • Variable-length input handling
  • Custom signature padding (\xBE\xEE)
  • Compact, educational output
  • Easy to extend and experiment with
